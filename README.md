# React Homework Project

This repository contains my homework assignments for the React course.

## Homework 1

### Overview
In this homework, I set up a basic React project using Vite (JavaScript template) and implemented a class component.

### Changes:
- Initialized React project with Vite.
- Cleaned up boilerplate code (removed default styles and assets).
- Updated `package.json` with project information and author details.
- Created a `List` class component that renders an arbitrary list of items from its state.
- Integrated the `List` component into `App.jsx`.

### How to Run:
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
